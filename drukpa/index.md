# 竹巴噶举（Drukpa Kagyu）

## 概述

竹巴噶举是藏传佛教噶举派的重要支系之一，属于达波噶举系统。其名称中的“竹”（'brug）意为雷龙，因此又被称为“雷龙传承”。

今天的竹巴噶举广泛弘传于西藏、拉达克、不丹、尼泊尔等地，其中不丹更以“竹域（Druk Yul，雷龙之国）”闻名于世。

## 传承源流

印度：

帝洛巴 → 那若巴

西藏：

玛尔巴 → 米拉日巴 → 冈波巴 → 帕竹噶举 → 竹巴噶举

## 创始人

竹巴噶举创始人为藏巴甲热·益西多吉（1161－1211）。

相传他在寻找建寺地点时，见九条龙腾空而起，因此以“竹（雷龙）”命名此传承。

## 三大特点

### 大手印

以直接认识心性为核心。

### 那若六法

包括拙火、梦观、光明、中阴等重要修法。

### 苦修传统

历代竹巴祖师十分重视闭关、实修与证悟。

## 三大支系

### 上竹巴

以郭仓巴系统最为著名。

### 中竹巴

以热龙寺法统为中心。

### 下竹巴

以洛热巴系统为代表。

## 不丹

十七世纪，夏仲阿旺朗杰进入不丹，统一诸部，建立政教体系。

自此竹巴噶举成为不丹最重要的佛教传统之一。

## 当代传承

现今竹巴噶举仍然活跃于喜马拉雅地区，并积极推动教育、环保与慈善事业。
